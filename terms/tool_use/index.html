Araç Kullanımı

term_id: tool_use

Category: application_paradigms

Definition

Araç Kullanımı, dil modellerinin önceden tanımlanmış işlevleri (hesap makinesi, arama motorları veya veritabanı sorguları gibi) çağırarak harici yazılım ortamlarıyla etkileşim kurmasını sağlar. Bu yaklaşım, modellerin yeteneklerini genişletir.

Summary

Yapay zeka ajanlarının, yerel yeteneklerinin ötesinde belirli görevleri yerine getirmek için harici işlevleri veya API’leri seçip yürüttüğü bir paradigmadır.

Key Concepts

  • İşlev Çağırma
  • Ajan Mimarisi
  • API Entegrasyonu
  • Harici Bilgi

Use Cases

  • Otomatik müşteri destek botları
  • Veri analizi asistanları
  • Akıllı ev otomasyonu
