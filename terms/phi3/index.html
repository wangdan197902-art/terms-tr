Phi3

term_id: phi3

Category: basic_concepts

Definition

Phi-3, Microsoft tarafından piyasaya sürülen küçük dil modelleri (SLM) serisidir; daha büyük modellerle karşılaştırılabilir yüksek performans sunarken önemli ölçüde daha az hesaplama kaynağı gerektirmeyi amaçlar.

Summary

Microsoft Araştırmaları tarafından geliştirilen açık ağırlıklı büyük dil modelleri ailesi.

Key Concepts

  • Küçük Dil Modelleri
  • Microsoft Araştırmaları
  • Kenar Yapay Zekası (Edge AI)
  • Sentetik Veri Eğitimi

Use Cases

  • Mobil cihazlarda veya dizüstü bilgisayarlarda LLM’leri çalıştırma
  • Kurumsal uygulamalar için maliyet etkin çıkarım
  • Düşük gecikme gereksinimleri olan gerçek zamanlı işleme

Related Terms

  • On-device AI (Cihaz İçi Yapay Zeka)
