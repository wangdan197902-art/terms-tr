Örneklemeye Programlama

term_id: programming_by_example

Category: basic_concepts

Definition

Örneklemeye Programlama (PBE), geliştiricilerin açık kod yazmak yerine somut girdi-çıktı çiftleri aracılığıyla istenen davranışı belirttiği program sentezi paradigmasıdır. Yapay zeka sistemi, bu örnekleri analiz ederek arka planda çalışan algoritmayı veya kodu oluşturur.

Summary

Kullanıcıların sağladığı girdi-çıktı örneklerinden kurallar çıkararak programların otomatik olarak sentezlendiği bir yapay zeka tekniği.

Key Concepts

  • Program Sentezi
  • Girdi-Çıktı Örnekleri
  • Tümdengelimli Çıkarım
  • Düşük-Kod/Sıfır-Kod

Use Cases

  • Örnek verilerden Excel formülü üretimi
  • Metin dönüşümü ve veri temizleme betikleri
  • Tekrarlayan kullanıcı arayüzü etkileşimlerinin otomasyonu

Related Terms

  • program_synthesis (program sentezi)
  • inductive_programming (tümdengelimli programlama)
  • few_shot_learning (az örnekli öğrenme)
  • code_generation (kod üretimi)
