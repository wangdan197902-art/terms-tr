ReAct

term_id: react

Category: application_paradigms

Definition

ReAct çerçevesi, Dil Modellerinin (LLM) hem akıl yürütme izleri hem de görev odaklı eylemleri iç içe geçmiş bir şekilde üretemesini sağlar. İnsan benzeri düşünce süreçlerini simüle ederek modellerin genişletilmiş araçlarla etkileşime girmesine olanak tanır.

Summary

ReAct, büyük dil modellerinin karmaşık görevlerdeki performansını artırmak için akıl yürütme (reasoning) ve eylem (acting) süreçlerini birleştiren bir istem (prompting) paradigmasıdır.

Key Concepts

  • İç İçe Akıl Yürütme
  • Eylem Üretimi
  • Gözlem Entegrasyonu
  • Düşünce Zinciri (Chain-of-Thought)

Use Cases

  • Karmaşık soru-cevap sistemleri
  • Otonom ajanlar
  • Gerçeklik kontrolü iş akışları

Related Terms

  • Chain-of-Thought (Düşünce Zinciri)
  • Agent (Ajan/Akıl Yürüten Sistem)
  • Prompt Engineering (İstem Mühendisliği)
  • LLM (Büyük Dil Modeli)
