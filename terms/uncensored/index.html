Sansürsüz

term_id: uncensored

Category: basic_concepts

Definition

Yapay zeka bağlamında ‘sansürsüz’, genellikle yerleşik güvenlik hizalamalarını kaldırmak veya zayıflatmak amacıyla ince ayar veya değişikliklere tabi tutulan modelleri tanımlar. Bu modeller, kullanıcıların güvenlik kısıtlamalarını atlamasına olanak tanır.

Summary

Güvenlik filtrelerini ve etik yönergeleri aşmak için değiştirilmiş yapay zeka modellerini ifade eder.

Key Concepts

  • Güvenlik Hizalaması
  • İnce Ayar (Fine-tuning)
  • İçerik Denetimi
  • Model Değişikliği

Use Cases

  • Model zafiyetlerini araştırma
  • Kısıtlama olmaksızın yaratıcı yazarlık
  • Güvenlik sınırlarını test etme

Related Terms

  • Jailbreaking (Hapis cezasından kaçırma/sistem yetkilerini ele geçirme)
  • Alignment (Hizalama)
  • RLHF (İnsan Geri Beslemesi ile Pekiştirmeli Öğrenme)
  • Safety Filters (Güvenlik Filtreleri)
